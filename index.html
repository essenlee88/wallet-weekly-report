App 侧：存量用户粘性增强与获客挑战并存，需警惕平台流量基础收缩

整体：App 总 UV 连续四周下降，W21 降至 1,425,292（WoW -7.09%，较 W19 峰值累计跌 16%）。结合关键漏斗产品（如 Markets、Trenches、Token Detail Page）UV 普降，判断这不是单一产品问题，而是平台整体用户访问规模收缩。然而，核心存量用户活跃度显著提升，Homepage 各榜单 Visits/User 增幅高达 +24.95% ~ +49.2%，表明用户沉浸深度增加，形成了"用户基数在流失，但留存用户使用更频繁"的典型存量博弈特征。

分产品：

  • Homepage 板块：功能粘性分化。Trending榜单 Clicks/User 达 5.08（+8.09%），作为首页核心引流位表现稳健。收藏榜单用户粘性极高（V/U 13.37，Clicks/User 36.45），但 UV -5.91%，提示核心收藏功能面临用户流失。
  • Markets 板块：Stock榜单数据修复带来"虚假繁荣"。其 CTR（25.88%，+25.38pp）与 Clicks/User（4.75，+124.06%）的暴涨，主要源于 W19-W20 疑似埋点问题导致基数过低后的"回归正常"，并非真实业务爆发。Markets 整体 UV -13.72%，Trending 子版块 UV -18.75%，表明该板块新用户获取或召回能力减弱，是整体 UV 下滑的主因之一。
  • Trenches 板块：Meme 类产品进入平台期。Meme Rush UV 连续四周下滑（-21.3%），但 CTR 稳定在 ~28%，说明产品对到访用户仍有吸引力，但用户新鲜感减退或外部引流减少导致拉新乏力。Meme Rank 在低基数下实现 CTR 反弹（31.94%，+5.32pp），可作为迭代优化的观察样本。Topic Rush UV 降幅最大（-17.71%），需关注主题运营的可持续性。
  • 落地页：流量质量两极分化。Stocks 榜单 CTR 高达 87.8%（+9.53pp），结合其 UV +10.33%，指向该页承接了高精准度的外部流量，转化效率极佳。Pre-IPO 榜单 UV 锐减 39%，表明市场热点或宣传周期已过。新上线的 hookSummer 主题榜单首周 UV 5,971，需后续观察留存。

💡 行动建议

  1. 【数据/PM 本周】诊断 UV 下滑根因：启动用户路径分析，定位连续 UV 下滑起点（新用户减少 vs 老用户流失），与同期市场数据（BTC 价格、链上活跃度）交叉验证，区分"市场普跌"与"产品问题" — App Total UV 较 W19 峰值累计跌 16%
  2. 【FE/数据 本周】确认 Stock 榜单埋点是否已修复：W19-W20 CTR 异常低（0.5%~0.78%），W21 回升至 25.88%，需 FE 确认事件是否在 W21 正确上报，避免将修复前数据误用于趋势判断 — App Markets > Stock CTR WoW +25.38pp
  3. 【PM 下期跟进】评估 Meme Rush 拉新策略：UV 4 周连续下滑（-21.3% 累计），但 CTR ~28% 稳定，说明到访用户仍有转化意愿，问题在引流端而非产品体验 — 可考虑新主题活动或外部导流渠道测试
  4. 【运营 本周】复盘 hookSummer 首周表现：首周 UV 5,971，CTR 60.12%，CTR 偏高可能因为是精准推送流量，需确认曝光量级和流量来源，判断能否持续 — W21 为首周，缺乏 WoW 基准
