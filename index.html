Web 侧：异常流量 W19 仍持续，Total UV / Token Detail Page / WoW 均不可信

⚠️ 数据污染说明（W18–W19 持续）

以下指标受异常流量影响，不建议直接使用或对外引用：

  • Web Homepage Total UV（15.1 万）：较 W16/W17 正常基线（5.6–7.6 万）仍偏高约 2 倍；VPU 从正常的 11–13 骤降至 6.14，符合 bot 单页访问特征
  • Token Detail Page UV（9.9 万）：单页 UV 占平台总 UV 的 66%，严重超出正常比例，是异常流量的主要来源
  • Web 全部产品的 UV WoW：分母（W18 Homepage Total）本身有污染，WoW 计算失真
  • Penetration（渗透率）：分母为被污染的 Total UV，渗透率数值失真
  • Visits/User（人均访问次数）：被大量单页 bot 访问拉低，不反映真实用户行为

相对可信的指标（受 UV 分母影响较小）：

  • CTR、Clicks/User、Token CTR、Token C/U — 若产品 UV 本身无异常，点击深度指标仍有参考价值
  • Trenches（Meme Rush 等）UV 量级较小（约 1–2 万），异常流量影响有限，数据相对可信

分板块（仅供参考，需结合污染背景）：

  • Markets：Markets Trending UV 9.3 万量级可疑（与 Token Detail Page 同量级），CTR 0.33% 偏低，不排除含 bot 流量
  • Trenches：Meme Rush UV 9,843（+3.5%），Meme Rank / Topic Rush 小幅增长，量级正常，数据相对可信
  • Tokenised Stock：UV 1,902（+43.7%），量级小但增速高，深度指标良好，值得关注
