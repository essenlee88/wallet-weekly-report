App 侧数据解读

整体趋势总结：本周平台总UV稳中有升，Homepage核心榜单（Trending、Holdings）渗透率与用户粘性（Visits/User）显著增长，但Token Detail Page遭遇大幅下滑（UV -24.2%，渗透率 -4.7pp），显示用户从浏览到深度研究的转化链路出现断裂。转化效率方面，多数榜单CTR与Token CTR呈分化态势：Homepage榜单普遍微增，而Markets及Trenches板块多数榜单CTR环比下降，用户点击意愿减弱。

  • Trenches / Meme 板块：Meme Rush 与 Meme Rank 用户量小幅增长（+2.5% / +2.3%），但渗透率微降，CTR分化明显——Meme Rank 的 Token CTR 逆势提升 +1.8pp 至 29.8%，而 eMeme (Event Rush) 虽 UV 暴增 41.9%，CTR 却骤降 16.2pp，表明流量激增但用户精准度下降，需优化事件匹配机制。
  • 榜单板块（Homepage & Markets）：Homepage 各榜单（Trending、收藏、Holdings、Stock）UV 全面正增长，其中 Stock 榜单 UV 飙升 47.8%，渗透率 +0.9pp，CTR 提升 4.6pp 至 30.4%，成为 Homepage 转化效率提升最快的入口。Markets 板块的 Stock 榜单同样亮眼，UV 增长 58.4%，CTR 提升 3.4pp 至 59.2%，用户对“股票类”代币关注度显著上升。
  • Portfolio / 组件板块：My Position 组件 UV 增长 14.8%，渗透率 +0.2pp，但 CTR 与 Trade CTR 均下滑（-1.0pp / -0.7pp），用户持仓查看意愿增强，但交易转化动力不足，建议增加一键交易或价格提醒功能提升转化。
  • 落地页板块：Stocks 榜单 UV 增长 26.0%，渗透率 +0.4pp，CTR 提升 2.3pp 至 48.6%，表现稳健；但主题类落地页（CN Meme、hookSummer、X402、Pre-IPO）UV 普遍大幅下滑（-39.2% ~ -75.2%），热度消退明显，需评估主题生命周期并提前规划新热点。

重点产品关注（3-5个）：

  • Homepage > Stock 榜单：UV +47.8%，CTR +4.6pp 至 30.4%，渗透率 +0.9pp。可能原因：市场对“股票映射代币”兴趣升温，建议持续强化该榜单的实时行情与交易入口，并考虑引入更多传统金融标的。
  • Markets > Stock 榜单：UV +58.4%，CTR +3.4pp 至 59.2%，转化率全榜最高。可能原因：用户从 Homepage 进入后，在 Markets 页面有更强的比价与筛选需求，建议优化榜单排序逻辑，增加“涨幅/成交量”等筛选维度。
  • Token Detail Page：UV -24.2%，渗透率 -4.7pp，CTR -4.5pp。可能原因：用户浏览深度下降，或首页直接跳转交易路径被其他入口分流。建议分析用户流失节点，考虑增加“相关代币推荐”或“一键交易”按钮提升留存。
  • Trenches > eMeme (Event Rush)：UV +41.9%，但 CTR 骤降 16.2pp。可能原因：活动引流质量不高，用户被吸引进入但内容匹配度低。建议优化事件标签与用户兴趣画像的匹配算法，或增加“参与即得”激励提升点击。
  • 落地页 > Stocks 榜单：UV +26.0%，CTR +2.3pp 至 48.6%，且无交易点击率数据（N.A.），说明用户以浏览为主。建议增加“一键交易”按钮或“加入自选”功能，将浏览行为转化为交易或收藏行为。
