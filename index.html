App 侧：股票板块全面领涨，核心资产页深度交互下滑需警惕

整体：App 总 UV 在连续三周下滑后，W22 反弹至 153.5 万（+7.71% WoW），但 PV 增幅（+1.6%）远小于 UV，导致人均访问次数下降至 23.79（-5.67%）。用户总规模回升，但整体深度浏览/交互意愿在减弱。

股票板块：本周最突出的增长引擎，全线强劲。Homepage > Stock榜单 UV 大增 47.78%，CTR 提升 4.56pp；Markets > Stock榜单 UV 飙升 58.45%，CTR 也增长 3.4pp；落地页 > Stocks榜单 UV 增长 26.03%。「股票代币化」产品对 App 用户吸引力巨大，用户沿主页→Markets→落地页全链路深度探索。

主页/榜单：主页核心榜单（Trending、Holdings、收藏）UV 均稳健增长（+2%~+10%），基础活跃度良好。Markets > Perps榜单 UV 增长 16.65%，但 CTR（5.96%）偏低且环比大跌 5.88pp，需关注榜单内容与用户预期的匹配度。

落地页：冰火两重天。通用 Stocks榜单 表现优异（+26%），但时效性主题榜单（CN Meme -39%、hookSummer -75%、X402 -55%、Pre-IPO -63%）UV 全线暴跌，主题活动热度周期已过，用户兴趣迅速消退。

Trenches：Meme Rush / Meme Rank / Topic Rush UV 增幅均微弱（+1%~+2.5%），进入平稳期，当前玩法对拉新或促活效果可能已触达天花板。

Token Detail Page：关键负面信号。TDP UV 连续四周下滑（W19: 29.7万 → W20: 25.2万 → W21: 22.6万 → W22: 17.1万），本周环比锐减 24.23%，CTR 同步大跌 4.49pp。访问深度与交互效率双重恶化，可能影响用户的交易决策与持仓管理体验。

💡 行动建议

  1. [数据/PM]【立即】 诊断 TDP（Token Detail Page）衰退根源 — UV 连续 4 周下滑至 17 万，本周再跌 24%；交叉分析跳出率、CTA 点击、加载时长，确认是流量分发问题还是页面体验问题
  2. [PM]【本周】 巩固股票板块优势 — Homepage / Markets Stock 双双爆发（+47%/+58%），分析高增长用户画像，考虑加大 Markets 页曝光或设计股票代币新用户引导
  3. [运营/PM]【下期跟进】 审视 Perps 榜单策略 — UV 增长但 CTR 骤降 5.88pp，优化榜单选品规则或视觉呈现，测试能否提升点击意愿
  4. [运营]【下期跟进】 清理过气主题榜单 — hookSummer / X402 / Pre-IPO UV 暴跌超 50%，考虑下线或替换为新热点主题，避免占用曝光位
