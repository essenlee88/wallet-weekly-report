App 侧：核心页面 UV 普遍下滑，首页人均交互韧性凸显，需优先诊断数据缺失问题

⚠️ 数据说明：本周（W21）App 端 8 个 Dashboard 数据采集超时失败，涉及 App/Markets 板块（Trending / Alpha / 收藏 / Holding / Stock 共 5 个榜单）、App/Trenches（Meme Rush、Meme Rank、Topic Rush）。Trending 榜单 CTR、收藏榜单 CTR 及上述缺失模块数据不可用，分析均已清晰标注。

整体：过去 4 周 App 核心指标持续承压。① 总 UV 连续收缩：App Homepage UV 自 W19 峰值（169.6 万）持续下滑，W21 降至 142.5 万（-7.1%），TDP UV 同步 4 周连降；② 人均粘性（V/U）逆势回升：Homepage V/U W21 升至 25.22（+5.3%），Trending V/U 飙升 49%，说明存量用户访问深度在增强；③ 转化漏斗承压：TDP CTR 连续 4 周下降，W21 仅 18.39%（较 W18 -2.74pp）；④ 数据质量警报：Stock 榜单在 W19-20 CTR 暴跌至 <1% 后，W21 突然"恢复"至 25.88%，可信度存疑，必须核实埋点。

分产品：

  • Homepage 矩阵：Trending 榜单 UV 连续微降（W21 130.7 万，-2.9%），V/U 异常飙升 49% 至 5.61，可能是新交互功能上线或少量用户反复浏览，CTR 数据丢失是分析阻碍；Holdings 榜单表现最稳，CTR 稳定在 20.05%；Stock 榜单 CTR 突然恢复 25.88%（+25.38pp，Clicks/U +124%），因 W19-20 历史异常存疑，需优先验证埋点。
  • Markets & Trenches（W21 数据全部缺失）：Markets Page UV 连续 4 周下跌，W21 已降至 25.3 万（-13.7%），是当前最大流量流失区；Trenches 三板块（Meme Rush / Meme Rank / Topic Rush）在 W18-20 呈稳健态势，本周因缺失无法评估最新趋势。
  • Tracker：Signal/Sentiment UV W21 -15.4% 至 1.9 万，CTR -1.9pp 至 26.28%，Trade CTR 仅 8.59%，信号至交易的转化效率是核心瓶颈。
  • TDP：形势最严峻。UV 连续 3 周下滑（W21 22.5 万，-10.7%），CTR 连续 4 周下降（W21 18.39%，-2.45pp WoW），来访减少的同时转化意愿也在降低。
  • 落地页：Stocks 榜单是本周亮点——UV +10.3%，CTR 高达 87.8%（+9.53pp），强主题驱动转化能力突出；Pre-IPO 榜单却连续 3 周大幅下滑（W21 3,710，-39%），需结合市场热点判断原因。

💡 行动建议

  1. 【立即/本周】 [数据] 诊断 8 个 App Dashboard 采集超时根因 — W21 数据完全缺失，Markets + Trenches 均无法评估，判断是 Sensors 查询超时还是管道故障，并补采 — App/Trenches Meme Rush W20→W21 趋势断层
  2. 【立即/本周】 [FE/数据] 验证 Stock 榜单 CTR 数据真实性 — W19-20 CTR 跌至 <1% 后 W21 突然反弹至 25.88%，需回放 W19-20 埋点日志，确认是功能中断还是上报中断 — CTR 波动达 25pp，决策依赖数据正确性
  3. 【立即/本周】 [PM] 深入分析 TDP 双降问题 — UV -10.7% + CTR -2.45pp 连续下滑，核查 TDP 流量来源分布变化（首页/搜索/外链各自占比），以及"交易按钮"曝光率是否有 UI 遮挡问题
  4. 【产品建议】 [PM] 参考 Stocks 落地页高转化模式，评估 CTR 提升机制能否复用至 TDP 或主页榜单 — Stocks 落地页 CTR 87.8% vs TDP 18.4%，两者设计差异值得研究
