关键结论
  • 总量对比：STON.fi 7d $188M，与 Uniswap V2 ETH（$135M）量级相当；但 Uniswap V2+V3+V4 合计 $5.3B，差距 28 倍。
  • 新 token 活跃度：ETH Uniswap 新 pool 24h 活跃量（$4.6M+，不完整）约为 STON.fi 新 pool（~$9.4K）的 约 500 倍，ETH 新 token 生态远比 TON 活跃。
  • 结构差异：STON.fi 总量中主流对占比高，ETH Uniswap V2 新 token 占比更高但绝对体量仍远超 STON.fi；V3/V4 以蓝筹对为主，新 token 占比低。
  • 小结：TON 链新 token 交易规模约为 ETH 的 1/500，当前接入对交易量贡献有限；Telegram 入口的唯一性是长期战略价值点。
