MemeRank 权重优化方案

目标：提升非 FLAP token 在 Top 榜单中的占比，鼓励优质 BSC 链上活跃项目

核心调整思路

降权
Market Cap（30% → 15%）：MC 与 FLAP 项目高度相关，过高权重导致大市值 FLAP 长期占据榜首。降至 15% 保留基本代表性，同时为 BN 活跃指标让出空间。
提权
BN 交易者（5% → 21%）+ BN 交易笔数（5% → 18%）：反映 Binance 生态真实活跃用户数量与频次，对 FLAP 项目有较强区分度。BN 总权重 66%
提权
Token 上线时间（2.5% → 5%）：创建时间越近得分越高，自然补偿新 token 在 BN 数据积累不足的劣势，无需额外惩罚机制。
调整
BN Holders（7.5% → 7%）：保留现有口径，鼓励项目方向 BN 用户空投，适度降低比重防止纯刷持有人。
* BN 信号总权重从初始 42% 提升至 66%；P24h / P7d / Holders / Liquidity / Impression 小幅下调为活跃指标让路。
