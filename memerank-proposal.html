MemeRank 权重优化 Proposal

模拟数据：2026-05-12 快照（150 tokens）· 目标：通过排名机制激励项目方导入 Binance 钱包真实交易流量

需求背景

FLAP 地推盘高度关注 MemeRank 排名，但对 Binance 钱包实际使用严重不足—— 持有者多来自线下发放，而非 BN Swap 主动买入，BN Traders 极少。
策略：大幅提升 BN Traders / BN Vol / BN Tx 在 MemeRank 中的权重， 让项目方为维持排名必须引导用户用 BN 钱包真实交易，排名机制成为 BN 生态使用率的杠杆。

⚖️ 设计原则

列顺序 = 权重优先级

前端 UI 从左到右：市值 → B.成交量 → B.交易者·B.笔数 → B.持有地址 → 价格% → 流动性。权重大小与展示顺序对齐。

橙色列 = 核心指标

UI 中 B.成交量、B.交易者、B.笔数、B.持有地址 四列橙色高亮。BN 指标总权重 48% → 70%，提升 22pp。

Traders 是核心杠杆

直接衡量「有多少人用 BN 钱包主动交易」。从 5% 提升至 20%（×4），是激励目标最直接的指标。
