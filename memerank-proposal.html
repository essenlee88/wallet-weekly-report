MemeRank 权重优化方案

数据来源：2026-05-12 快照（150 tokens）· 目标：提升 BN 钱包使用激励，尤其是 Traders 和 Tx 指标

核心逻辑：FLAP 地推盘重视 MemeRank 但 BN 钱包使用不足。大幅提升 BN 交易者 + 交易笔数权重，让项目方需要真实驱动 BN 交易才能维持高排名。
BN 信号权重合计：42.5% → 61.0%（+18.5pp）

关键变化说明

BN 交易者BN Traders：5% → 21%（+16pp）
最强激励杠杆：直接衡量通过 BN 钱包真实交易的用户数，对 FLAP 地推盘区分度最高。
BN 交易笔数BN Tx：5% → 15%（+10pp）
衡量 BN 钱包活跃频次，鼓励项目方持续驱动真实交易行为。
BN 成交量BN Vol：25% → 20%（-5pp）
适度降低，将权重让位给行为指标（交易者数 + 笔数）。
BN 持有地址BN Holders：8% → 5%（-2pp）
当前部分项目刷 BN 持有地址迹象明显，适度降权；保留口径鼓励项目方向 BN 用户空投。
币龄：2% → 5%（+2pp）
新 token（< 7 天）createTime 得分 5/5，老 token（> 90 天）平均仅 1.35/5，提权重给新币自然加分，无需额外逻辑。
市值：30% → 17%（-13pp）
保留 17% 确保上榜 token 具备基本市值门槛，避免市值极低的项目进入排名。
