渗透率指标说明
BN 渗透率 = 在所有链上交易该 token 的用户中，通过 Binance Web3 Wallet 交易的用户占比。
渗透率高 → BN 用户是主要交易群体；渗透率低 → 大量链上交易者绕过 BN 在其他平台交易。
渗透率 = min( BN_7d_traders / 链上_7d_traders, 100% )
rate_score = decile_rank(渗透率) × 0.5 // 0.5 ~ 5.0
渗透率贡献 = rate_score × 10%
